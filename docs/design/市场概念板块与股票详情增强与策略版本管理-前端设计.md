### 3.3 全局股票搜索组件（stock/search）

#### 组件：`StockSearchAutocomplete`

```
src/components/stock-search-autocomplete/stock-search-autocomplete.tsx
src/components/stock-search-autocomplete/index.ts
```

**功能**：提供全局可复用的股票模糊搜索下拉组件。

**布局**：

```
┌─────────────────────────────────┐
│  🔍 输入股票代码或名称...       │
├─────────────────────────────────┤
│  000001.SZ  平安银行  银行      │
│  000002.SZ  万科A     房地产    │
│  600000.SH  浦发银行  银行      │
│  ...                            │
└─────────────────────────────────┘
```

**交互**：

1. 用户输入关键词（≥ 1 字符）→ debounce 300ms → 调用 `stockApi.search({ keyword, limit: 20 })`
2. 下拉展示搜索结果，每行显示：`tsCode` | `name` | `industry`
3. 选中后触发 `onChange(item: StockSearchItem)` 回调
4. 可选：选中后导航到股票详情页

**实现要点**：

- 基于 MUI `Autocomplete` + `TextField`
- 使用 `useMemo` + `useCallback` 做 debounce
- Loading 状态时显示 `CircularProgress`

**Props**：

```typescript
type StockSearchAutocompleteProps = {
  /** 选中回调 */

  /** 当前值 */
  value?: StockSearchItem | null;
  /** placeholder */
  placeholder?: string;
  /** 是否全宽 */
  fullWidth?: boolean;
  /** 额外的 sx */
  sx?: SxProps;
};
```

**使用场景**：

- 可在全局 header 搜索栏嵌入（后续）

**布局**：

```
┌──────────────────────────────────────────────────────────────┐

├──────────────────────────────────────────────────────────────┤
│  ┌───────────────────────┬───────────────────────┐           │
│  │ v2 (2026-04-10)       │ v3 (2026-04-12)       │           │
│  │ 变更说明: ...          │ 变更说明: ...          │           │
│  └───────────────────────┴───────────────────────┘           │
├──────────────────────────────────────────────────────────────┤
│  配置差异列表                                                 │
│  ┌──────────┬──────────┬──────────┬──────────┐               │
│  │ 变更类型 │ 配置路径 │ 旧值(vA) │ 新值(vB) │               │
│  ├──────────┼──────────┼──────────┼──────────┤               │
│  │ CHANGED  │ stop_loss│  -5%     │  -8%     │               │
│  │ ADDED    │ rotation │  —       │ { ... }  │               │
│  │ REMOVED  │ filter_x │ true     │  —       │               │
│  └──────────┴──────────┴──────────┴──────────┘               │
├──────────────────────────────────────────────────────────────┤
│                                              [关闭]          │
└──────────────────────────────────────────────────────────────┘
```

**交互**：

1. Dialog 打开时 → 调用 `compareStrategyVersions({ strategyId, versionA, versionB })`
2. 顶部显示两个版本的基本信息（版本号 + 时间 + changelog）

   - `changeType` 列使用 `Label` 颜色区分：`ADDED` → success，`REMOVED` → error，`CHANGED` → warning

   - `oldValue` / `newValue` 使用 `JSON.stringify(value, null, 2)` 展示，超长时截断 + Tooltip 显示完整值
4. 无差异时显示 "两个版本配置完全一致" 提示
