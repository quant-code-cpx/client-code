# 行业轮动分析（Industry Rotation）— 前端实现规划

> **目标读者**：AI 代码生成助手。请严格按照本文定义的组件结构、布局规则、API 调用方式实现。

---

## 一、技术栈与约定

与市场概览页一致，参见 `市场概览-前端设计.md` 中"技术栈与约定"部分。

**额外提醒**：

1. **API 前缀**：行业轮动后端端点均为 `POST /api/industry-rotation/*`，使用 `apiClient.post<T>()` 调用。
2. **日期参数**：多数接口接受 `trade_date`（YYYYMMDD 格式）和 `period` 时间窗口参数，不传则后端取最新交易日。
3. **金额单位**：资金流向数据单位为**万元**（Tushare 原始单位），页面展示统一转为"亿元"（÷10000），保留 2 位小数。
4. **涨跌颜色**：涨/正值用 `error.main`（红色），跌/负值用 `success.main`（绿色），中性用 `text.secondary`。
5. **空数据保护**：每个组件使用数据前必须检查 null / undefined / 空数组，防止图表渲染错误。
6. **加载状态**：每张卡片独立管理 `loading` 状态，使用 MUI `Skeleton` 占位。

## 五、各组件详细规范

### 5.1 `RotationOverviewCards` — 行业轮动总览（概要卡片组）

**API**：`POST /api/industry-rotation/overview`
**请求体**：`{ trade_date?: string, period?: string }`

**预期响应结构**：

```typescript
type RotationOverviewResult = {
  tradeDate: string;
  period: string;
  topGainers: Array<{ name: string; pctChange: number }>; // 领涨行业 Top 3
  topLosers: Array<{ name: string; pctChange: number }>; // 领跌行业 Top 3
  topInflows: Array<{ name: string; netAmount: number }>; // 资金净流入 Top 3
  avgPctChange: number; // 全行业平均涨跌幅
  riseCount: number; // 上涨行业数
  fallCount: number; // 下跌行业数
  totalCount: number; // 行业总数
};
```

**UI 结构**：

- 横向 4 张统计卡片（`Grid` xs=6 md=3）：
  - **卡片 1** — 行业涨跌比：`riseCount / fallCount`，附圆形进度指示器
  - **卡片 2** — 全行业平均涨跌幅：`avgPctChange`（大字，红涨绿跌）
  - **卡片 3** — 领涨行业：列出 Top 3 名称 + 涨跌幅
  - **卡片 4** — 领跌行业：列出 Top 3 名称 + 涨跌幅
- 每张卡片使用 `Card` + `CardContent`，标题用 `Typography variant="subtitle2"`，数字用 `Typography variant="h4"`

### 5.2 `RotationHeatmapChart` — 行业轮动热力图

**API**：`POST /api/industry-rotation/heatmap`
**请求体**：`{ trade_date?: string, period?: string }`

**预期响应结构**：

```typescript
type RotationHeatmapResult = {
  tradeDate: string;
  sectors: Array<{
    name: string;
    pctChange: number;
    amount: number; // 成交额（万元）
    netAmount: number; // 净流入金额（万元）
    children?: Array<{
      // 子行业（可选）
      name: string;
      pctChange: number;
      amount: number;
    }>;
  }>;
};
```

**UI 结构**：

- 卡片标题："行业热力图"
- 使用 ApexCharts `treemap` 类型
  - 块大小：按 `amount`（成交额）决定
  - 块颜色：按 `pctChange` 映射，红色（涨）→ 灰色（0）→ 绿色（跌）
  - 块标签：行业名称 + 涨跌幅百分比
- tooltip 显示：行业名称、涨跌幅、成交额（亿）、净流入（亿）
- 右上角提供模式切换 Toggle：按涨跌幅着色 / 按净流入着色

**ApexCharts 配置要点**：

```typescript
{
  chart: { type: 'treemap', toolbar: { show: false } },
  plotOptions: {
    treemap: {
      enableShades: false,
      colorScale: {
        ranges: [
          { from: -10, to: -3, color: '#00B8D9' },   // 大跌 → info
          { from: -3, to: -0.5, color: '#22C55E' },   // 小跌 → success
          { from: -0.5, to: 0.5, color: '#919EAB' },  // 平盘 → grey
          { from: 0.5, to: 3, color: '#FFAB00' },     // 小涨 → warning
          { from: 3, to: 10, color: '#FF5630' },       // 大涨 → error
        ],
      },
    },
  },
  dataLabels: { enabled: true, style: { fontSize: '12px' } },
  tooltip: { shared: false, intersect: true },
}
```

**交互**：点击某个行业块 → 调用 `onSectorClick(name)` → 打开 `RotationDetailDrawer`。

### 5.3 `RotationMomentumChart` — 行业动量排名

**API**：`POST /api/industry-rotation/momentum-ranking`
**请求体**：`{ trade_date?: string, period?: string, limit?: number }`

**预期响应结构**：

```typescript
type MomentumRankingResult = {
  tradeDate: string;
  period: string;
  rankings: Array<{
    name: string;
    momentum: number; // 动量值（收益率 %）
    rank: number;
    prevRank: number; // 上期排名
    rankChange: number; // 排名变动（正=上升）
  }>;
};
```

**UI 结构**：

- 卡片标题："行业动量排名"
- 右上角：数量限制下拉 `limit`（10 / 15 / 20 / 全部）
- 水平条形图（ApexCharts `bar` + `horizontal: true`）
  - Y 轴：行业名称，按 `momentum` 降序排列
  - X 轴：动量值（%）
  - 颜色：正值红色，负值绿色
  - 每行末尾标注排名变化箭头（↑ 绿色 / ↓ 红色 / — 灰色），使用 `annotations` 或自定义 `dataLabels`

**ApexCharts 配置要点**：

```typescript
{
  chart: { type: 'bar', toolbar: { show: false } },
  plotOptions: {
    bar: {
      horizontal: true,
      barHeight: '60%',
      colors: {
        ranges: [
          { from: -100, to: 0, color: '#22C55E' },
          { from: 0, to: 100, color: '#FF5630' },
        ],
      },
    },
  },
  xaxis: { labels: { formatter: (val: number) => `${val.toFixed(2)}%` } },
  tooltip: { shared: false, intersect: true },
}
```

**交互**：点击某行业条 → `onSectorClick(name)` → 打开详情。

### 5.4 `RotationReturnComparisonChart` — 行业收益对比

**API**：`POST /api/industry-rotation/return-comparison`
**请求体**：`{ trade_date?: string, period?: string, sectors?: string[] }`

**预期响应结构**：

```typescript
type ReturnComparisonResult = {
  period: string;
  benchmark: { name: string; data: Array<{ tradeDate: string; cumReturn: number }> };
  sectors: Array<{
    name: string;
    data: Array<{ tradeDate: string; cumReturn: number }>;
  }>;
};
```

**UI 结构**：

- 卡片标题："行业收益对比"
- 顶部行业多选芯片组（`Chip` + `Autocomplete`），最多选 6 个行业对比，默认展示总览的领涨/领跌各 2 个
- 折线图（ApexCharts `line`）：
  - X 轴：日期（datetime）
  - Y 轴：累计收益率（%）
  - 每条线代表一个行业，颜色区分
  - 增加一条灰色虚线代表基准（沪深300）
  - tooltip 显示日期 + 各行业累计收益

**ApexCharts 配置要点**：

```typescript
{
  chart: { type: 'line', toolbar: { show: false }, zoom: { enabled: true } },
  stroke: { curve: 'smooth', width: 2 },
  xaxis: { type: 'datetime' },
  yaxis: { labels: { formatter: (val: number) => `${val.toFixed(2)}%` } },
  tooltip: { shared: true, intersect: false },
  legend: { position: 'top' },
}
```

### 5.5 `RotationFlowAnalysisChart` — 行业资金流转分析

**API**：`POST /api/industry-rotation/flow-analysis`
**请求体**：`{ trade_date?: string, period?: string }`

**预期响应结构**：

```typescript
type FlowAnalysisResult = {
  tradeDate: string;
  period: string;
  flows: Array<{
    name: string;
    netInflow: number; // 净流入（万元）
    inflowAmount: number; // 流入总额
    outflowAmount: number; // 流出总额
    inflowRatio: number; // 流入占比（%）
  }>;
  topInflowSectors: string[]; // 资金流入 Top N
  topOutflowSectors: string[]; // 资金流出 Top N
};
```

**UI 结构**：

- 卡片标题："行业资金流转"
- 右上角 Toggle：图表模式（柱状图 / 气泡图）
- **柱状图模式**（默认）：

  - 分组柱状图（ApexCharts `bar`）
  - X 轴：行业名称（按净流入排序）
  - 系列 1（红色）：流入金额（亿），系列 2（绿色）：流出金额（亿），系列 3（蓝色线）：净流入
  - 默认显示 Top 15 净流入 + Bottom 15 净流出

- **气泡图模式**（可选）：
  - X 轴：涨跌幅，Y 轴：净流入，气泡大小：成交额
  - 便于观察资金与价格的关系

**ApexCharts 配置要点（柱状图）**：

```typescript
{
  chart: { type: 'bar', toolbar: { show: false }, stacked: false },
  plotOptions: { bar: { columnWidth: '50%' } },
  stroke: { show: true, width: 2, colors: ['transparent'] },
  yaxis: { labels: { formatter: (val: number) => `${(val / 10000).toFixed(1)}亿` } },
  tooltip: { shared: true, intersect: false },
}
```

type ValuationResult = {

**UI 结构**：

  - Y 轴：行业名称（按分位数升序 = 低估在上）
  - X 轴：分位数（0~100%）

    - 30~70%：`warning.main`（黄色 = 中性）

  - 每行右侧标注当前 PE/PB 具体数值
- 下方可选表格视图（`Table`），展示完整数据含当前值 + 中位数 + 分位数

### 5.7 `RotationDetailDrawer` — 单行业详情

**API**：`POST /api/industry-rotation/detail`
**请求体**：`{ sector_name: string, trade_date?: string, period?: string }`

**预期响应结构**：

```typescript
type RotationDetailResult = {
  sectorName: string;
  tradeDate: string;
  // 基础指标
  pctChange: number;
  amount: number;
  netAmount: number;
  momentum: number;
  pePercentile: number;
  pbPercentile: number;
  // 收益走势
  returnTrend: Array<{ tradeDate: string; cumReturn: number; benchmarkReturn: number }>;
  // 资金流向走势
  flowTrend: Array<{ tradeDate: string; netInflow: number; cumulativeInflow: number }>;
  // 成分股 Top N
  topStocks: Array<{
    tsCode: string;
    name: string;
    pctChg: number;
    mainNetInflow: number;
    amount: number;
  }>;
};
```

**UI 结构**：

- 使用 MUI `Drawer`（`anchor="right"`，宽度 `480px`，md 以上 `640px`）
- **头部**：行业名称（`Typography variant="h5"`）+ 关闭按钮 + 当日涨跌幅标签
- **指标卡片区**（2×3 Grid）：
  - 涨跌幅、成交额（亿）、净流入（亿）、动量值、PE 分位、PB 分位
  - 分位数显示带颜色标签（低估/中性/高估）
- **收益走势 Tab**：
  - 折线图：行业累计收益 vs 基准（沪深300）
  - ApexCharts `line`，双线对比
- **资金流向 Tab**：
  - 柱线复合图：柱 = 每日净流入，线 = 累计净流入
- **成分股排行**：
  - `Table` 展示 Top 10 个股：名称、涨跌幅、主力净流入、成交额
  - 点击个股可跳转 `/stock/detail/:tsCode`

**触发方式**：在热力图 / 动量排名 / 估值分位中点击某行业名称时打开。

## 八、交互与联动说明

### 8.1 全局筛选器联动

页面视图组件维护 `tradeDate` 和 `period` 两个状态，传递给所有子组件。任一筛选器变更时，所有子组件重新请求数据。

```tsx
const [tradeDate, setTradeDate] = useState<string>('');
const [period, setPeriod] = useState<string>('1m');
```

### 8.2 组件间联动

| 触发                          | 动作                                                     |
| ----------------------------- | -------------------------------------------------------- |
| 热力图点击行业块              | 打开 `RotationDetailDrawer`                              |
| 动量排名点击行业条            | 打开 `RotationDetailDrawer`                              |
| 估值分位点击行业条            | 打开 `RotationDetailDrawer`                              |
| 总览卡片点击领涨/领跌行业名称 | 打开 `RotationDetailDrawer`                              |
| 收益对比默认行业              | 从总览 API 的 `topGainers` + `topLosers` 前 2 名自动填入 |
| 详情抽屉内点击个股            | 路由跳转 `/stock/detail/:tsCode`                         |

### 8.3 错误处理

- 每个组件独立 try/catch，失败时显示 `Alert` 提示而非阻断整页。
- API 超时或网络错误时，显示"加载失败，点击重试"按钮。
