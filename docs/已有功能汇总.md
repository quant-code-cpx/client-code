### 2.1 认证与登录

**路由**：`/sign-in` · **Section**：`src/sections/auth/`

| 功能             | 说明                                                             |
| ---------------- | ---------------------------------------------------------------- |
| 图形验证码登录   | 获取图形验证码（SVG），输入账号 + 密码 + 验证码登录              |

| 角色权限         | 支持 SUPER_ADMIN / ADMIN / USER 三级角色                         |
