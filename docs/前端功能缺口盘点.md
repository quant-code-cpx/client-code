## 三、部分覆盖的模块（端点级缺口）

### Backtest — 回测（前端 10/18，覆盖 56%）

#### ❌ 未接入的端点

| 后端端点                                   | 说明                       |
| ------------------------------------------ | -------------------------- |
| `POST /backtests/walk-forward/runs`        | 创建 Walk-Forward 验证任务 |
| `POST /backtests/walk-forward/runs/list`   | Walk-Forward 验证列表      |
| `POST /backtests/walk-forward/runs/detail` | Walk-Forward 验证详情      |
| `POST /backtests/walk-forward/runs/equity` | OOS 拼接净值曲线           |
| `POST /backtests/comparisons`              | 创建多策略对比回测组       |
| `POST /backtests/comparisons/detail`       | 对比组详情                 |
| `POST /backtests/comparisons/equity`       | 对比净值曲线叠加           |
| `POST /backtests/rolling/runs`             | 滚动窗口回测               |

**建议**：Walk-Forward、多策略对比、滚动窗口是策略评估的高级功能，对量化研究极为重要，优先级 **高**。

### Factor — 因子（前端 9/21，覆盖 43%）

#### ❌ 未接入的端点

| 后端端点                               | 说明                      |
| -------------------------------------- | ------------------------- |
| `POST /factor/analysis/orthogonalize`  | 因子正交化                |
| `POST /factor/analysis/fama-macbeth`   | Fama-MacBeth 截面回归检验 |
| `POST /factor/custom/create`           | 创建自定义因子            |
| `POST /factor/custom/test`             | 试算表达式                |
| `POST /factor/custom/update`           | 更新自定义因子            |
| `POST /factor/custom/delete`           | 删除自定义因子            |
| `POST /factor/custom/precompute`       | 触发单因子预计算          |
| `POST /factor/admin/precompute`        | 手动触发因子预计算        |
| `POST /factor/admin/backfill`          | 触发历史因子值回补        |
| `POST /factor/admin/precompute/status` | 查询预计算状态            |
| `POST /factor/backtest/submit`         | 因子策略一键回测          |
| `POST /factor/backtest/attribution`    | 因子归因分析              |

**建议**：

- 自定义因子引擎（create/test/update/delete/precompute）是因子研究闭环的关键，优先级 **高**。
- 因子一键回测和归因分析与 Strategy/Backtest 联动，优先级 **高**。
- Fama-MacBeth 检验和正交化分析属于高级功能，优先级 **中**。
- Admin 接口（precompute/backfill/status）可纳入 Tushare 管理面板，优先级 **低**。
