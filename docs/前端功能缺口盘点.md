## 二、完全缺失的模块（0% 前端覆盖）

### 🚨 Strategy — 策略管理（8 端点）

后端已提供完整的策略模板 CRUD + 克隆 + 发起回测 + JSON Schema 获取，前端没有对应的 section、API 文件、路由和导航入口。

| 后端端点                   | 说明                                   |
| -------------------------- | -------------------------------------- |
| `POST /strategies/create`  | 创建策略模板                           |
| `POST /strategies/list`    | 查询策略列表（按类型/标签/关键词过滤） |
| `POST /strategies/detail`  | 策略详情                               |
| `POST /strategies/update`  | 更新策略（config 变更时版本号自增）    |
| `POST /strategies/delete`  | 删除策略                               |
| `POST /strategies/clone`   | 克隆策略（支持克隆公开策略）           |
| `POST /strategies/run`     | 基于策略发起回测                       |
| `POST /strategies/schemas` | 获取所有策略类型的 JSON Schema         |

**建议**：Strategy 是回测工作流的核心入口（用户创建/管理策略 → 发起回测），优先级 **高**。

### 🔶 Heatmap — 市场热力图（1 端点）

| 后端端点             | 说明                 |
| -------------------- | -------------------- |
| `POST /heatmap/data` | 市场涨跌幅分布热力图 |

**建议**：可嵌入市场概览页，优先级 **中**。

### 🔶 Index — 指数模块（3 端点）

| 后端端点                   | 说明               |
| -------------------------- | ------------------ |
| `POST /index/list`         | 支持的核心指数列表 |
| `POST /index/daily`        | 指数日线行情       |
| `POST /index/constituents` | 指数成分股及权重   |

**建议**：前端 Market 模块已通过 `/market/index-quote` + `/market/index-trend` 展示指数概览，但 Index 模块的日线详情和成分股权重尚未接入，可嵌入市场概览或独立页面，优先级 **中**。

## 三、部分覆盖的模块（端点级缺口）

### Backtest — 回测（前端 10/18，覆盖 56%）

#### ❌ 未接入的端点

| 后端端点                                   | 说明                       |
| ------------------------------------------ | -------------------------- |
| `POST /backtests/walk-forward/runs`        | 创建 Walk-Forward 验证任务 |
| `POST /backtests/walk-forward/runs/list`   | Walk-Forward 验证列表      |
| `POST /backtests/walk-forward/runs/detail` | Walk-Forward 验证详情      |
| `POST /backtests/walk-forward/runs/equity` | OOS 拼接净值曲线           |
| `POST /backtests/comparisons`              | 创建多策略对比回测组       |
| `POST /backtests/comparisons/detail`       | 对比组详情                 |
| `POST /backtests/comparisons/equity`       | 对比净值曲线叠加           |

**建议**：Walk-Forward、多策略对比、滚动窗口是回测系统的高级功能，对策略评估极为重要，优先级 **高**。

### Strategy-Draft — 策略草稿箱（API 6/6，但无独立页面）

前端已有 `strategy-draft.ts` API 文件完整对接 6 个端点，但仅嵌入回测工作台使用，没有独立的 section、路由和导航入口。

**建议**：当 Strategy 模块前端实现后，策略草稿箱应作为其子页面呈现，优先级随 Strategy 模块走。
